# Plant Seedling Classification using CNN

## Overview
This project builds a Convolutional Neural Network (CNN) to classify plant seedling images into multiple categories. The goal is to demonstrate an end-to-end deep learning workflow for image classification.

## Objectives
- Prepare and preprocess image data
- Build a CNN model for multi-class classification
- Train and validate the model
- Evaluate model performance
- Present a clean portfolio-ready computer vision project

## Tools and Libraries
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- scikit-learn

## Project Workflow
1. Data loading and exploration
2. Image preprocessing and resizing
3. Train-validation split
4. CNN model development
5. Model training and validation
6. Performance evaluation
7. Final conclusions and improvement areas

## Results
The CNN successfully learned discriminative image features for plant seedling classification. The project demonstrates the practical use of deep learning for image recognition tasks.

## Key Skills Demonstrated
- Computer Vision
- Deep Learning
- CNN Architecture Design
- Model Evaluation
- Python for Machine Learning
